## Описание

Десктопное приложение для управления магазином с ролями пользователей:

- Клиент (просмотр товаров)
- Менеджер (просмотр + фильтрация + заказы)
- Админ (полный CRUD: товары и заказы)
- Гость (только просмотр товаров)

UI реализован на **PyQt6**, база данных — **MySQL**, доступ к БД — **PyMySQL**.

---

## Технологии

- Python 3.10+
- PyQt6
- MySQL
- PyMySQL

## Роли пользователей 
## Клиент 
- Просмотр товаров без взаимодействия с товарами


## Менеджер
- Поиск товаров
- Фильтрация и сортировка
- Просмотр заказов (без редактирования)


## Админ
## Полный CRUD:

- Добавление товаров/заказов
- Редактирование
- Удаление
- Подсветка выбранных карточек
- Работа с заказами через JOIN (user/product)


## Гость

- Только просмотр товаров

---

## Логика авторизации

После входа по логину и паролю пользователь направляется в окно по роли:

client > ClientWindow

manager > ManagerWindow

admin > AdminWindow


гость не может авторизоваться, он переходит по кнопке "Войти как гость"

guest > ClientWindow (read-only)
